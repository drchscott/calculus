Notice here that the derivative function requires computing a limit using the value \(f(x)\text{.}\) So, the domain of the derivative is, at most, the domain of \(f\text{.}\) Indeed, if we look at the function \(f(x)=|x|\text{,}\) we see from its graph that \(|x|'\) is \(1\) for \(x \gt 0\) and \(|x|'\) is \(-1\) for \(x \lt 0\text{.}\) But what about at \(x=0\text{?}\) If we calculate the definition of \(|x|'\) at \(x=0\) using \(h \to 0^{+}\text{,}\) we get \(1\) but using \(h \to 0^{-}\text{,}\) we get \(-1\text{.}\) So, for \(|x|\text{,}\) the derivative doesnâ€™t exist at \(x=0\text{.}\) Of course, again, looking at the graph you can see there is a "point" or "cusp" at \(x=0\) so there is no well defined tangent line there. So, we should have expected a failure to find the derivative (i.e. slope of the tangent line) there.
