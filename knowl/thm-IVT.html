Theorem 2.3.1.

If \(f\) is continuous on an interval \(I\) (whether open, half open, or closed) and \(r\) and \(s\) are two values that \(f\) obtains and finally \(w\) happens to be a number between (i.e. intermediate to) \(r\) and \(s\text{,}\) then there is some \(x\) in \(I\) so that \(f(x)=w\text{.}\)
