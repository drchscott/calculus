For the moment, let’s stay focused on the simple example from the previous section (i.e. \(f(x)=x\) on \([0,2]\)). First, we note that the interval we’re working over runs from \(x=0\) to \(x=2\text{.}\) So, it has size 2-0 = 2. Chopping that into n equal subintervals means that each subinterval will have size (1/n)th the size of the whole interval so each one will have size \(\frac{2}{n}\text{.}\) Now that we know the size of the subintervals and we also know that the first one starts at the left end of the whole interval (i.e. at 0 in this case), we see that the subintervals we’re using are
\begin{equation*} [0,\frac{2}{n}],\quad [\frac{2}{n},2\frac{2}{n}],\quad [2\frac{2}{n},3\frac{2}{n}],\quad \dots ,\quad [(n-1)\frac{2}{n},n\frac{2}{n}] \end{equation*}
Thus, the right endpoints of the subintervals where we’ll be evaluating \(f(x)=x\) in order to get the heights of our rectanges are
\begin{equation*} \frac{2}{n}, 2\frac{2}{n}, 3\frac{2}{n}, \dots ,n\frac{2}{n} \end{equation*}
So, finally, we can compute
\begin{equation*} \begin{array}{lll} R_n \amp = \amp \frac{2}{n}f(\frac{2}{n}) + \frac{2}{n}f(2\frac{2}{n}) + \frac{2}{n}f(3\frac{2}{n}) + \dots + \frac{2}{n}f(n\frac{2}{n}) \\ \amp = \amp \frac{2}{n}(\frac{2}{n} + 2\frac{2}{n} + 3\frac{2}{n} + \dots + n\frac{2}{n}) \\ \amp = \amp \frac{2}{n}(\frac{2}{n}(1 + 2 + 3 + \dots + n))\\ \amp = \amp (\frac{2}{n})^2(1 + 2 + 3 + \dots + n) \end{array} \end{equation*}
