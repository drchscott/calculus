Theorem 1.3.8.

  1. If \(a\) is any number, then \(\displaystyle \lim_{x\to a}sin(x)=sin(a)\text{.}\)
  2. If \(a\) is any number, then \(\displaystyle \lim_{x\to a}cos(x)=cos(a)\text{.}\)
  3. If \(a\) is any number, then \(\displaystyle \lim_{x\to a}e^x=e^a\text{.}\)
  4. If \(a > 0\text{,}\) then \(\displaystyle \lim_{x\to a}ln(x)=ln(a)\text{.}\)
