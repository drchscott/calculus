    Section 5.3 Definition

    Riemann’s solution from the previous section suggests that we would get more and more accurate estimates if we computed \(R_{10}, R_{100}, R_{1000}, \) etc. That way we’d be using thinner and thinner approximating rectangles. But, each time we do a Riemann sum, it’s a lot of work. So, let’s try something. Suppose we don’t specify exactly how many rectangles to use. Just let \(n\) denote a positive number so that \(R_n\) is denoting the Riemann sum for that many rectanges. As \(n\) gets larger, the rectangles should get thinner and \(R_n\) should be a better and better approximation to the area. Can we get \(R_n\) written without specifying the exact value of \(n\text{?}\) Well, let’s try.
    For the moment, let’s stay focused on the simple example from the previous section (i.e. \(f(x)=x\) on \([0,2]\)). First, we note that the interval we’re working over runs from \(x=0\) to \(x=2\text{.}\) So, it has size 2-0 = 2. Chopping that into n equal subintervals means that each subinterval will have size (1/n)th the size of the whole interval so each one will have size \(\frac{2}{n}\text{.}\) Now that we know the size of the subintervals and we also know that the first one starts at the left end of the whole interval (i.e. at 0 in this case), we see that the subintervals we’re using are
    \begin{equation*} [0,\frac{2}{n}],\quad [\frac{2}{n},2\frac{2}{n}],\quad [2\frac{2}{n},3\frac{2}{n}],\quad \dots ,\quad [(n-1)\frac{2}{n},n\frac{2}{n}] \end{equation*}
    Thus, the right endpoints of the subintervals where we’ll be evaluating \(f(x)=x\) in order to get the heights of our rectanges are
    \begin{equation*} \frac{2}{n}, 2\frac{2}{n}, 3\frac{2}{n}, \dots ,n\frac{2}{n} \end{equation*}
    So, finally, we can compute
    \begin{equation*} \begin{array}{lll} R_n \amp = \amp \frac{2}{n}f(\frac{2}{n}) + \frac{2}{n}f(2\frac{2}{n}) + \frac{2}{n}f(3\frac{2}{n}) + \dots + \frac{2}{n}f(n\frac{2}{n}) \\ \amp = \amp \frac{2}{n}(\frac{2}{n} + 2\frac{2}{n} + 3\frac{2}{n} + \dots + n\frac{2}{n}) \\ \amp = \amp \frac{2}{n}(\frac{2}{n}(1 + 2 + 3 + \dots + n))\\ \amp = \amp (\frac{2}{n})^2(1 + 2 + 3 + \dots + n) \end{array} \end{equation*}
    We take a moment now to note some facts about sums like the one that shows up here. Namely,

    Lemma 5.3.1. Special Sums.

    1. \(\displaystyle 1 + 2 + \dots + n = \frac{n(n+1)}{2}\)
    2. \(\displaystyle 1^2 + 2^2 + \dots + n^2 = \frac{n(n+1)(2n+1)}{6}\)
    3. \(\displaystyle 1^3 + 2^3 + \dots + n^3 = \frac{(n(n+1))^2}{4}\)
    We present these summation facts without proof here. But, please note, they are fairly straightforward to verify. One can find many nice treatments online and generally if this text is being used for teaching a course, one should expect to see some presentation of one or more verifications in class. The first of them is particularly evident and is often attributed to the hisotric luminary mathematician, Karl Gauss.
    With these summation facts in place, we can continue the work above.
    \begin{equation*} \begin{array}{lll} R_n \amp = \amp (\frac{2}{n})^2(1 + 2 + 3 + \dots + n) \\ \amp = \amp (\frac{2}{n})^2\frac{n(n+1)}{2} \\ \amp = \amp 2\frac{n(n+1)}{n^2} \\ \amp = \amp 2\frac{n}{n}\frac{n+1}{n} \\ \amp = \amp 2(1+\frac{1}{n}) \\ \end{array} \end{equation*}
    So, if we want \(R_{100}\text{,}\) it’s super easy now as we see that it is \(2(1+\frac{1}{100}) = 2.02\text{.}\) Even more, it is now very clear that the more and more (thinner and thinner) rectangles we use, the more this is looking like \(2\text{.}\)

    Definition 5.3.2. Definition of Integral.

    For a continuous function \(f\) definied on a closed interval \([a,b]\text{,}\) we define
    \begin{equation*} \int_{a}^{b} f(x)dx = \displaystyle \lim_{n\to \infty} R_n \end{equation*}
    At the risk of being more verbose that consistent with the goals of this text, let’s put down what \(R_n\) looks like in general. So, for a function \(f(x)\) on an interval \([a,b]\) we see that the size of the interval is \(b-a\) so that when we divide it into \(n\) equal size subintervals, each has size \((b-a)/n\text{.}\) Since our interval starts at \(a\text{,}\) we see that the first subinterval ends at \(a+(b-a)/n\text{,}\) the second subinterval ends at \(a+2(b-a)/n\text{,}\) the third one ends at \(a+3(b-a)/n\) and so on until we finally reach the end of the last, \(n\)th, subinterval at \(a+n(b-a)/n\text{.}\) We evaluate \(f(x)\) at each of these subinterval endpoints and multiply each of those evaluations by the size of the subintervals to get the area of each approximating rectangle and add those areas up to get \(R_n\text{.}\) In other words, the most general form of the Riemann sums is
    \begin{equation*} R_n = \frac{b-a}{n}\left[ f\left(a+\frac{b-a}{n}\right)+f\left(a+2\frac{b-a}{n}\right)+\dots+f\left(a+n\frac{b-a}{n}\right) \right] \end{equation*}
